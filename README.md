# Ton Bot - Telegram Miniapp

Sistema de referidos con TON, panel de admin, detección de VPN y pagos automáticos.

## Características
- ✅ Sistema de referidos 3 niveles
- ✅ Pagos automáticos en TON
- ✅ Detección de VPN y multicuentas
- ✅ Panel de administrador
- ✅ Historial de transacciones
- ✅ Top de referidos
- ✅ Multiidioma (EN/ES)
- ✅ Anuncios Monetag
